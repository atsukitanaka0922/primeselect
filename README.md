# Prime Select ECサイト

📝 概要
Prime Select は、PHPで構築された実務レベルのECサイトです。ユーザー認証、商品カタログ、ショッピングカート、チェックアウトプロセス、管理パネルなど、本格的なECサイトに必要な機能を実装しています。
このプロジェクトは、実際の実務環境で使用されるECサイトの構造とコード設計を学ぶことを目的としています。

🚀 インストール

リポジトリをクローンする
git clone https://github.com/atsukitanaka0922/primeselect.git

Webサーバーのドキュメントルートに配置する（またはローカル開発環境にコピーする）
# XAMPP の場合
cp -r ecommerce-php /path/to/xampp/htdocs/ecommerce

データベースのセットアップ

PHPMyAdminにアクセスし、ecommerce_dbという名前のデータベースを作成
database.sqlファイルを実行してテーブルを作成


データベース接続設定

config/database.phpファイルを開き、必要に応じてデータベース接続情報を変更


画像フォルダの作成と権限設定
mkdir -p assets/images
chmod 755 assets/images


🖥️ 使用方法
1. ユーザー側機能

ホームページ: すべての商品のプレビューを表示
商品一覧: カテゴリ別や価格範囲でフィルタリングして商品を表示
商品詳細: 商品の詳細情報、複数画像、レビューの表示
カート: 商品の追加、削除、数量変更
チェックアウト: 注文情報の入力、支払い方法の選択
マイページ: プロフィール更新、注文履歴、お気に入り商品の管理

2. 管理者機能

管理ログイン: /admin/login.phpからアクセス
ダッシュボード: 販売統計、最近の注文の表示
商品管理: 商品の追加、編集、削除
カテゴリ管理: カテゴリの追加、編集、削除
注文管理: 注文ステータスの更新、注文詳細の表示
ユーザー管理: ユーザー情報の管理
